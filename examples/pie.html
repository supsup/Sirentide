Sirentide — first diagram 🌊

A pie chart, rendered from a tiny DSL to static SVG at build time. No JavaScript, no runtime.

  • Reviews — 40
  • Builds — 25
  • Docs — 20
  • Design — 15

The DSL that made it

pie
  "Reviews" : 40
  "Builds"  : 25
  "Docs"    : 20
  "Design"  : 15

How

echo 'pie
  "Reviews" : 40
  "Builds"  : 25
  "Docs"    : 20
  "Design"  : 15' | java -jar build/libs/sirentide-0.1.0.jar > pie.svg

The SVG above is Sirentide's verbatim bake output — only <svg> and filled <path> elements (the wedges and the labels), no <script>, no <style>, no runtime. It's safe to inline into any page and survives a strict HTML sanitizer untouched.

Those slice labels render inside the diagram — each one is a run of glyph outlines drawn as SVG paths from the bundled STIX Two Math font (the same face LatteX renders formulas in), never a <text> element. So there are no web fonts and no view-time font dependency: the text is baked geometry, byte-identical everywhere. (The legend still gives the value breakdown.)
