Sirentide — first diagram 🌊

A pie chart, rendered from a tiny DSL to static SVG at build time. No JavaScript, no runtime.

  • Reviews — 40
  • Builds — 25
  • Docs — 20
  • Design — 15

The DSL that made it

pie
  "Reviews" : 40
  "Builds"  : 25
  "Docs"    : 20
  "Design"  : 15

How

echo 'pie
  "Reviews" : 40
  "Builds"  : 25
  "Docs"    : 20
  "Design"  : 15' | java -jar build/libs/sirentide-0.1.0.jar > pie.svg

The SVG above is Sirentide's verbatim bake output — only <svg> and filled <path> wedges, no <script>, no <style>, no runtime. It's safe to inline into any page and survives a strict HTML sanitizer untouched.

Slice labels live in this page's legend for now — Sirentide draws text as SVG paths (the next build: the glyph-outline reader), at which point the labels render inside the diagram, in the same STIX Two Math face LatteX renders formulas in.
