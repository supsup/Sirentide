Sirentide — bar chart 🌊

An xychart, from DSL to static SVG at build time. Axes, bars, and labels — all inert geometry.

← pie example

The DSL that made it

xychart
  "Mon" : 5
  "Tue" : 8
  "Wed" : 3
  "Thu" : 6
  "Fri" : 9

Everything you see is Sirentide's verbatim bake output — <line> axes, <rect> bars, and the category + value labels as glyph <path>s (STIX Two Math outlines, never <text>). No <script>, no <style>, no runtime. Bar heights are pure arithmetic (value ÷ max × plot height) — the tallest bar (Fri = 9) reaches the top. Same pipeline as the pie: DSL → IR → layout → SVG, second diagram type, zero new moving parts at view time.
